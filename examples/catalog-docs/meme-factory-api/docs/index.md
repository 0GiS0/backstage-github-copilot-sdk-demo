# Meme Factory API

Meme Factory API coordinates uploads, moderation rules, publication state, and access to shared meme assets.

## Responsibilities

- Create meme drafts
- Store and retrieve media assets through Meme Vault
- Expose the main HTTP contract used by the frontend and AI assistant

## Relationships

- Provides the Meme Factory HTTP API
- Depends on Meme Vault

## Owner

The component is maintained by the Meme Factory Team.
