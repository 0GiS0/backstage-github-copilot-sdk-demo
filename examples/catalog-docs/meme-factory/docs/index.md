# Meme Factory

Meme Factory is a compact demo system that shows how a single product can be modeled with a domain, a system, multiple components, one API, and one shared resource.

## Components

- Meme Factory Web
- Meme Factory API
- Meme Captioner

## Shared Resource

- Meme Vault

## Key Relationships

- The web frontend consumes the public API.
- The API and AI assistant both depend on Meme Vault for media storage.
- The whole system is owned by the Meme Factory Team.
