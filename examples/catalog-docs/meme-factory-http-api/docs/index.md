# Meme Factory HTTP API

This API exposes the core contract for Meme Factory clients.

## Endpoints

- `POST /memes` to create meme drafts from uploaded screenshots
- `POST /captions` to request caption suggestions
- `GET /health` for service health checks

## Consumers

- Meme Factory Web
- Meme Captioner

## Owner

The API belongs to the Meme Factory API component and is maintained by the Meme Factory Team.
