# Meme Captioner

Meme Captioner is the AI service responsible for turning screenshots and PR anecdotes into reusable caption suggestions.

## Responsibilities

- Generate captions from raw text and screenshot context
- Offer multiple tones for the same meme draft
- Reuse the central API contract for job submission and result retrieval

## Relationships

- Consumes the Meme Factory HTTP API
- Depends on Meme Vault

## Owner

The component is maintained by the Meme Factory Team.
