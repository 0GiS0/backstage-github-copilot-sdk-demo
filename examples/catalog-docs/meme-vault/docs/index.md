# Meme Vault

Meme Vault is the shared storage resource behind Meme Factory.

## What It Stores

- Raw screenshots uploaded by engineers
- Caption candidates produced by the AI assistant
- Final meme exports published by the backend

## Current Dependents

- Meme Factory API
- Meme Captioner

## Ownership

The resource is owned by the Meme Factory Team and belongs to the Meme Factory system.
