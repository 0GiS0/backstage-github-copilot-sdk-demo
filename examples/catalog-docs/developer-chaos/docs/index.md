# Developer Chaos

Developer Chaos is the umbrella domain for lightweight internal tools that help engineers survive noisy releases, confusing incidents, and pull-request fatigue.

## Scope

- Shared internal developer-experience utilities
- Team-facing workflows with clear ownership
- Demo entities used to showcase catalog relationships in Backstage

## Owned Systems

- Meme Factory

## Operating Model

The domain is owned by the Meme Factory Team and is intentionally small so the relationship graph stays easy to explain in demos.
